About ReadMe.com

A full-stack web application where customers can shop for a large variety of books. The main objective of the project is to create an online book store that allows users to search and purchase books online based on Title, Author, ISBN, Publisher, or Genre. Customers can add books to their cart for checkout or to their bookmarks to save for later, view their past orders, view their account information, and change their password from their profile. This project was created through the Revature training program.

    Technologies Used

  • Java
  • Spring Boot and Spring Data
  • JUnit
  • Jenkins
